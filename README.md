# OMEGA-PROTOCOL: MakeCode Battle Royale

## Installation
1. Open [MakeCode Arcade](https://arcade.makecode.com).
2. Click **Import** > **Import URL**.
3. Paste this repository link.

## Usage
- **A Button**: Jump from Bus / Fire Weapon.
- **B Button**: Toggle Perspective.
- **Menu**: View Inventory.
